= Accident_data = 

A few scripts to process a csv file of road traffic accident data.
The main aim was to be able to add administrative geography info to the csv file
http://www.uk-postcodes.com/ is used to perform the look up.

==Install==

The scripts are mean to be run via php_cli. 
Small tweaks will make them run on a webserver.
